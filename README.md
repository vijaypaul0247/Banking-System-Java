# Banking System - Java Project

---

## Project Description
This is a simple Banking System application developed using Core Java.  
The program allows users to create a bank account, deposit money, withdraw money, and check their account balance.

---

## Features
- Create bank account
- Deposit money
- Withdraw money
- Check account balance
- Exception handling for insufficient balance

---

## Technologies Used
- Java
- OOP Concepts
- Exception Handling
- Scanner Class

--- 

## How to Run the Project

1. Open the project in Eclipse / IntelliJ / VS Code
2. Compile the Java file
3. Run the program
4. Follow the menu options to perform banking operations

---

## Sample Menu

1. Deposit
2. Withdraw
3. Check Balance
4. Exit
